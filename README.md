#  Career Track (Organizador de Candidaturas)

O **Career Track** é um sistema web desenvolvido para automatizar e organizar o gerenciamento de candidaturas a vagas de emprego. O projeto foca em otimizar o acompanhamento de processos seletivos através de um painel de controle (dashboard) centralizado.

Desenvolvido com a arquitetura **MVC (Model-View-Controller)** e padrão **DAO (Data Access Object)**, o sistema garante uma separação clara de responsabilidades, código limpo e fácil manutenção.

## Funcionalidades
* **Gestão de Candidaturas (CRUD):** Inserção, consulta, atualização e exclusão de registros de vagas.
* **Dashboard de Acompanhamento:** Monitoramento do status de cada processo seletivo aplicado.
* **Persistência de Dados:** Integração segura e estruturada com banco de dados relacional.

## Tecnologias e Arquitetura
* **Front-end:** Java Server Pages (JSP)
* **Back-end:** Java (Servlets / Controllers)
* **Padrões de Projeto:** MVC e DAO
* **Banco de Dados:** MySQL
* **Ambiente/IDE:** NetBeans
